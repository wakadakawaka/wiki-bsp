//        ** ФизическоеЛицо - СправочникСсылка.ФизическиеЛица

//        ** Период - Дата
//        ** ОтработаноДнейКалендарных - Число

//     * ДанныеСтрахователей - ТаблицаЗначений - содержит колонки:
//        ** ФизическоеЛицо - СправочникСсылка.ФизическиеЛица
//        ** Год - Число
//        ** Страхователь - СправочникСсылка.Организации
