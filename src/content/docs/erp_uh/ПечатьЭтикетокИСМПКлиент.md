// Возвращает новую структуру параметров для использования в процедуре РаспечататьНовыйКодМаркировки.
//
// Параметры:
// 	СтруктураПечати - см. ПечатьЭтикетокИСМПКлиентСервер.СтруктураПечатиЭтикетки
