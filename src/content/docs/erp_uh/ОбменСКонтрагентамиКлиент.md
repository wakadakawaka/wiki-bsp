// Конвертирует переданные двоичные данные в произвольный электронный документ.
//

//
Процедура НачатьСозданиеЭлектронногоДокументаПоФайлу(ОповещениеОЗавершении, ПараметрыСоздания, ПараметрыФайла) Экспорт
